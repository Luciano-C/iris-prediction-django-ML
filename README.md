<h2>Installing requirements</h2>

\$ virtualenv venv <br>
\$ . venv/Scripts/activate <br>
\$ pip install -r requirements.txt <br>
\$ . venv/Scripts/activate <br>


<h2>Running Jupyter Notebook</h2>

\$ jupyter notebook <br>

\$ python manage.py runserver <br>
